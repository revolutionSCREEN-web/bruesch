# bruesch

Digitale Signage von Brüesch Gestaltungstechnik AG.

## glueckrad/
Das Brüesch Glücksrad für die Gewerbeausstellung (statische Web-App, offline-tauglich).
Live über GitHub Pages: **/bruesch/glueckrad/index.html**

- `index.html` – Anzeige-Seite (Signage, Hochformat)
- `form.html` – Handy-Formular für die Lead-Erfassung (QR-Ziel)
- `config.js` – Branding, Preise, Sensor, Timings, Lead-Erfassung
